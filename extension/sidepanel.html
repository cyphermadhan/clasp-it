Add Clasp It as an MCP server in Claude Code so it can read your picks. The MCP key and extension key must be the same.

1. Run in terminal (click to copy with your key):

claude mcp add --scope user --transport http clasp https://claspit.dev/mcp --header "Authorization: Bearer YOUR_KEY"

2. Restart Claude Code, then say: "fix all recent picks using clasp-it"

To update the key later:

claude mcp remove clasp

Then re-run step 1 with the new key.
